# Expense Tracker

**Автор:** Хасанова Аделина

## Описание
Приложение для отслеживания личных расходов с фильтрацией по категориям и дате, подсчётом суммы и сохранением данных в JSON.

## Как использовать
1. Запустите `main.py`.
2. Введите сумму, выберите категорию, укажите дату и нажмите "Добавить расход".
3. Для фильтрации выберите категорию и/или диапазон дат и нажмите "Применить фильтр".
4. Итоговая сумма отображается внизу таблицы.

## Примеры
- Добавить расход 500 ₽ на еду 2026-05-03.
- Фильтровать расходы по категории "Транспорт" с 2026-05-01 по 2026-05-10.
